# Frontend (Vue 3 + Vite + TypeScript)

Base frontend app shell for Gamified Study Hub.

## Scope in This Phase
Implemented:
- Vue 3 + Vite + TypeScript scaffold
- TailwindCSS setup
- Typed environment config (`VITE_API_BASE_URL`)
- Axios HTTP client placeholder
- Basic app shell page

Not implemented yet:
- Authentication flows
- Router-based feature pages
- Pinia stores
- Domain/business UI modules

## Local Run
```bash
npm install
npm run dev
```

## Quality Commands
```bash
npm run typecheck
npm run lint
npm run build
```
